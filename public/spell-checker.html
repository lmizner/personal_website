SPELL-CHECKER

A program in C that implements a spell checker by cross-checking words from a dictionary file with
words from a user specified text file. Words not found in the dictionary will be returned to the user
as misspelled words.

To run the program type "./main filename.txt" into the terminal window. The text files you can choose
from are "constitution.txt", "wordsworth.txt", and "lalaland.txt".
